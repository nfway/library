## 挑选领导者

### 首要：专长

这些人做过什么事情，有何专长。<br><br>
{% note warn For Mac users %}
应该问：你最擅长什么？<br>

少点问：你不具备什么素质，不擅长什么。
{% endnote %}
  

### 评估组织状况

这个机构/组织面临最直接的巨大的挑战是什么？

- 筹集经费？

- 重建士气？

把专长和组织的实际需求结合起来。

  

### 品质

一个人是否具备正直、诚实的品质？
{% note warn 选择什么样的人 %}
“我总是在问我自己，我是否愿意让自己的儿子在那个人（领导者）手下工作？如果他成功了，那么年轻人都会效仿他。我希望自己的儿子去效仿他么？”。所谓将心比心，不外如是。
{% endnote %}  

对于非营利组织的管理者而言，你必须学会平衡和统筹兼顾，综合考虑评价业绩的一系列标准

### 领导者能力

领导者应具备哪些能力？

1. 倾听的意愿、能力和自律。倾听不是技能，只不过需要自律；
2. 交流沟通的意愿，让其他人都能够理解你的意思，这需要极强的耐心；
3. 不回避问题。比如说“这件事没有做好，让我们重新再来，换个方法再做”。我们做事情不做则已，要做就要做得尽善尽美。
4. 个人服从工作需要的意愿，认识到与工作相比，个人的一切都显得无足轻重。

  

**关于决策** 

如果决策失误能否挽回？如果可以挽回，通常就可以接受更巨大的风险。<br><br>

 
{% note warn 关于机会 %}
对于愿意努力的人，如果第一次没有成功，那么领导者应给第二甚至第三次机会。<br>

对于不愿意努力的人，不要浪费精力。<br>
{% endnote %}  
如果要给新手压重担，提出严格的要求和更多的责任，那么必须为其安排一位导师来加以辅导指点。



### 领导者素质评估

1. 组织应对环境变化的调整能力如何？
2. 组织处理冲突的能力如何？
3. 组织满足利益相关者或客户等能力如何？
